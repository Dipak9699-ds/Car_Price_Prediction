# Car Price Prediction Using Machine Learning

A Machine Learning web application that predicts the price of a car based on features like:

* Car_Name
* Year
* Selling_Price
* Present_Price
* Kms_Driven
* Fuel_Type
* Seller_Type
* Transmission,Owner

# Problem Statement

Predict the price of a used car using machine learning regression techniques based on historical Indian car data.

# Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
